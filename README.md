## 📌 Project Overview

**Healora** is a modern medical tourism platform that simplifies the process of finding healthcare services abroad.

The platform allows patients to:

- Explore treatment options across multiple countries
- Compare treatment costs and recovery durations
- Discover experienced doctors and specialists
- Submit consultation requests directly through the website

The goal of this project was to build a **fully responsive healthcare platform using core web technologies without external frameworks.**

# ✨ Key Features

### 🏥 Treatment Exploration
Browse medical procedures across multiple countries with clear details about price, recovery time, and hospital information.

### 💰 Cost Comparison Tool
Compare treatment packages across hospitals and countries side-by-side using a dynamic JavaScript comparison system.

### 👨‍⚕️ Global Doctor Directory
Explore specialist doctor profiles with:

- Qualifications
- Experience
- Ratings
- Hospital affiliations

### 📩 Consultation Request System
Patients can submit consultation inquiries through a validated contact form.

### 📱 Fully Responsive Design
The entire website adapts smoothly to:

- Desktop
- Tablet
- Mobile devices

### ⚡ Interactive UI Enhancements
Includes modern web features such as:

- Smooth scrolling navigation
- Hover animations
- CSS transitions
- Keyframe animations
- Interactive filters
- Modal popups
